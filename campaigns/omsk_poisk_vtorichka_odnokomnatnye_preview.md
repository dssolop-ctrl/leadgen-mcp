- ❌ Квартиры от застройщика (новостроечный deny)

## Структура API (план создания)

1. `add_campaign` — кампания + 260 минус-слов + tracking_params (UTM на кампанию)
2. `add_sitelinks` × 1 — один набор быстрых ссылок (8 шт) на всю кампанию
3. `add_ad_extension` × 4-6 — уточнения (общий список)
4. Цикл по 11 кластерам:
   - `add_adgroup` — группа с region_ids=[66]
   - `add_keywords` — ключевые кластера
   - `update_autotargeting` — EXACT+ALTERNATIVE=YES, остальные NO
   - `add_ad` × 3 — 3 варианта объявления (A/B)
5. Кросс-минусация между группами (после создания всех групп)
6. `add_labels` — метки «Лидген, Вторичка, Покупатель»
