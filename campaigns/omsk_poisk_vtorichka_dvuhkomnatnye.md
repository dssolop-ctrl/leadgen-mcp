## Анализ и гипотезы (аналог — 706224151 «Омск | Поиск | Вторичка»)

Аналог: 27 602₽ за 30 дней → 1 909 кликов → 5 форм + 14 звонков. form_cpa 5 521, call_cpa 1 972. CTR и CPC приемлемые, но аналог очень широкий (вся вторичка в одной группе, одинаковые тексты). Ключевые проблемы:
- Единые заголовки "Купить квартиру в Омске" не различают комнатность
- Нет гео-групп (района) — трафик уходит в общий поток
- Текст про «квартиры от 950 тыс.» нерелевантен для двушек (их мин. цена ~2.3 млн)

**Что делаем в новой кампании:**
1. **Узкий скоуп** — только двухкомнатные. Это должно повысить CTR (запрос → объявление → посадочная все про 2-к).
2. **5 кластеров:** Общий (широкие) + 3 топ-района (Нефтяники/Московка/Левый берег) + Ценовой. Районы найдены через Wordstat SearchedWith (до 165 Shows/фраза).
3. **Заголовки с районом** для гео-групп ("Двушки на Московке, Омск" и т.д.) — гипотеза: более релевантные клики, снижение CPC.
4. **УТП «от 2.3 млн»** — адекватный ценовой якорь для двушек.
5. **Посадочная `/realty/dvuhkomnatnye-kvartiry/`** — отдельная страница для двушек, без фильтров по району (БД фильтров по Омску пуста — заполнить отдельной задачей).

## Стратегия

- **Цель:** Лиды на форму + реальные звонки через коллтрекинг
- **Пороги:**
  - CPA < 3 000₽ → масштабируем (+30% бюджет раз в 2 нед)
  - CPA 3 000–5 000₽ → наблюдаем, оптимизируем минус-слова и A/B
  - CPA > 5 000₽ → разбираем слабые группы, возможен возврат аналогу
- **План:**
  1. Запуск с WB_MAXIMUM_CONVERSION_RATE (10 000₽/нед)
  2. 2-3 недели обучения стратегии
  3. При 10+ конверсий/нед стабильно → переход на AVERAGE_CPA (target ~3 000₽, в чт-пт)
  4. Масштабирование через +30% бюджета или расширение ключевых по get_search_queries
