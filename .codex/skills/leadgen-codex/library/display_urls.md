# Отображаемые пути (DisplayUrlPath)

Короткий путь после домена, показываемый в объявлении вместо реального URL.
Увеличивает CTR за счёт релевантной подсказки.

> **Лимит Директа:** до 20 символов на путь.
> Разрешены: кириллица, латиница, дефисы, цифры.
> Запрещены: слеши, пробелы, query-параметры (? & =).

---

## Правила выбора

1. **Отражает суть объявления** — кластер, тип объекта, тематику.
2. **БЕЗ домена** (не повторять `omsk.etagi.com` в пути — Директ сам показывает домен).
3. **БЕЗ слешей** — «квартиры/омск» неверно, надо «Квартиры-Омск».
4. **С дефисами** вместо пробелов — «Купить-Однушку», не «Купить Однушку».
5. **Уникализируй по кластеру** — для гео-кластера путь отражает район.
6. Статический текст, **без макросов** типа `{keyword}`.

## Анти-паттерны

| ❌ НЕ так | ✅ Так | Почему |
|---|---|---|
| `omsk.etagi.com/realty` | `Квартиры-Омск` | Не повторяем домен |
| `квартиры омск` | `Квартиры-Омск` | Дефис вместо пробела |
| `купить квартиру омск вторичка ипотека` | `Купить-Квартиру` | Превышен лимит 20 символов |
| `?district=neftyaniki` | `Квартиры-Нефтяники` | Нет query-параметров |
| `Купить/квартиру` | `Купить-Квартиру` | Слеши запрещены |
| `{city}-жильё` | `Квартиры-{город}` | Макросы работают плохо и путают |

## Формула выбора

```
1. Определи основу: тематика + тип объекта (Купить-Квартиру / Дома-Коттеджи / Снять-Квартиру)
2. Добавь уточнение кластера (опционально):
   - Гео-кластер → + район (Квартиры-Нефтяники)
   - Ценовой → + «Недорого» (Квартиры-Недорого)
   - Финансовый → + «Ипотека» / «Рассрочка» (Квартира-в-Ипотеку)
3. Проверь лимит 20 символов. Если больше — сократи (убери «Купить», оставь ключевое).
4. Подставь в add_ad(display_url_path="...")
```
