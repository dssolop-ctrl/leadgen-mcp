# Продающие прибавки (selling modifiers)

Каноническая библиотека модификаторов по интент-типам. Используется:

- В **C4 Раунд 1** — для построения seed-матрицы (из каждого типа берём 1-2 фразы).
- В **C5** — для обязательного выделения интент-кластеров (каждый тип с ≥ 3 фраз становится отдельной группой).

> **Правило:** агент НЕ сворачивает все прибавки в 2 кластера («купить» и «недорого»).
> Минимум **4-5 интент-кластеров** на среднюю кампанию + гео-кластеры + общий.

---

## Тип 1. Прямой транзакционный интент

Пользователь готов купить, ищет прямо.

```
купить, покупка, приобрести, приобретение, оформить, оформление
```

**Примеры фраз для seeds:**
- купить квартиру
- купить однокомнатную
- купить двушку
- приобрести квартиру
- покупка квартиры
- оформить покупку квартиры

**Акценты заголовка кластера:** «Купить квартиру в {город}», «Квартиры для покупки».

---

## Тип 2. Ценовой интент

Пользователь ищет выгоду, сравнивает цены.

```
недорого, дешево, дешевая, дешевые, цена, цены, стоимость,
бюджетно, выгодно, выгодные, от ... т.р., от ... млн
```

**Примеры фраз для seeds:**
- купить квартиру недорого
- двушка недорого
- дешёвые квартиры
- квартира цена
- квартиры от 1 млн

**Акценты заголовка:** «Квартиры в {город} недорого», «Квартиры от 1.5 млн ₽», «Выгодные квартиры {город}».

---

## Тип 3. Финансовые условия

Пользователь думает о способе оплаты.

```
в ипотеку, ипотека, в рассрочку, рассрочка, с маткапиталом,
с материнским капиталом, в кредит, без первоначального взноса,
с господдержкой, семейная ипотека, сельская ипотека
```

**Примеры фраз для seeds:**
- купить квартиру в ипотеку
- квартира в рассрочку
- квартира с маткапиталом
- квартира без первоначального взноса
- семейная ипотека квартира

**Акценты заголовка:** «Квартиры в ипотеку», «Квартиры в рассрочку — {город}», «Семейная ипотека».

> ⚠️ При создании такого кластера проверь, что посадочная ведёт на страницу с калькулятором ипотеки или раздел «Ипотека» — иначе кластер неэффективен.

---

## Тип 4. Каталог / выбор

Пользователь в стадии изучения предложений.

```
объявления, варианты, каталог, база, выбор, подборка, список
```

**Примеры фраз для seeds:**
- объявления квартир
- квартиры варианты
- каталог квартир
- база квартир

**Акценты заголовка:** «Актуальная база квартир в {город}», «Более N квартир в каталоге», «Свежие объявления».

---

## Тип 5. Условия продавца / качество сделки

Пользователь хочет избежать посредников, ищет прямого продавца.

```
от хозяина, от собственника, без посредников, без комиссии,
от владельца, напрямую, первые руки
```

**Примеры фраз для seeds:**
- квартира от хозяина
- купить квартиру от собственника
- квартира без посредников
- квартиры от владельцев

**Акценты заголовка:** «Квартиры от собственников», «Без посредников — {город}».

> ⚠️ **«Без комиссии»** — только для тематик, где комиссия реально 0% (новостройки, коммерческая аренда некоторых типов). Для вторички — **ЗАПРЕЩЕНО** (см. `library/extensions.md` allow/deny).

---

## Тип 6. Источники / конкуренты (высокий интент покупателя)

Пользователь уже ищет на другом сайте, но можем перехватить.

```
авито, циан, домклик, яндекс недвижимость, mir kvartir, мир квартир,
n1, домофонд, sber недвижимость
```

**Примеры фраз для seeds:**
- авито квартиры
- циан квартиры
- домклик квартиры

**Акценты заголовка:** нейтральный «Квартиры в {город} — Этажи», **без прямого упоминания бренда-конкурента** (модерация режет).

> ⚠️ Название конкурента — только в ключевой фразе, не в Title объявления. Модерация Яндекса режет упоминание чужих брендов.

---

## Тип 7. Срочность / состояние объекта

Специфические запросы.

```
срочно, срочная продажа, готовая, готовые, новая, новые,
с ремонтом, с отделкой, с мебелью, под чистовую, под ключ,
свежая, свежие
```

**Примеры фраз для seeds:**
- срочно квартира
- готовая квартира
- квартира с ремонтом
- квартира под ключ

**Акценты заголовка:** «Готовые квартиры {город}», «Квартиры с ремонтом», «Заезжай и живи!».

---

## Правило Шага C5 (кластеризация)

**Для КАЖДОЙ кампании при наличии фраз проверяй каждый тип из библиотеки:**

| Тип | Условие создания кластера |
|---|---|
| 1. Транзакционный | ≥ 5 фраз ИЛИ ≥ 1 фраза с Shows ≥ 100 |
| 2. Ценовой | ≥ 3 фразы ИЛИ ≥ 1 фраза с Shows ≥ 50 |
| 3. Финансовый | ≥ 3 фразы ИЛИ ≥ 1 фраза с Shows ≥ 50 |
| 4. Каталог | ≥ 3 фразы ИЛИ ≥ 1 фраза с Shows ≥ 50 |
| 5. Условия продавца | ≥ 3 фразы ИЛИ ≥ 1 фраза с Shows ≥ 30 |
| 6. Источники | ≥ 3 фразы ИЛИ ≥ 1 фраза с Shows ≥ 30 |
| 7. Срочность | ≥ 3 фразы ИЛИ ≥ 1 фраза с Shows ≥ 30 |

**Минимум интент-кластеров в кампании: 4.** Если после C4 получается < 4 — вернись в C4 и добери семантику по недостающим типам через `check_search_volume`.

Дополнительно к интент-кластерам в кампании могут быть:
- Гео-кластеры (районы/микрорайоны)
- Объектные кластеры (однушки, двушки, студии — если кампания шире одного типа)
- Мини-кластер «Уточнение города» (5-10 фраз с упоминанием города)
- Catch-all «Общий» (широкие фразы без уточнений)

---

## Анти-паттерны (чего не делать)

- ❌ Сворачивать всё в 2 кластера: «Купить» и «Недорого». Теряем 5 интент-типов.
- ❌ Забывать про финансовый кластер — в недвижимости он самый горячий (ипотека, рассрочка).
- ❌ Не проверять источники/конкуренты — там прямой переток от Авито/Циан.
- ❌ Создавать кластер на 1 фразу — не даст статистики, засорит структуру.
