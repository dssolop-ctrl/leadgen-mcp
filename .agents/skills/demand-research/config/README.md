# Конфигурация demand-research

## Зависимости

Скилл demand-research **не требует собственного API-токена**. Все данные получаются через существующие инструменты:

### MCP Wordstat (настроен в .mcp.json)
- `check_search_volume` — быстрая проверка частотности фраз
- `get_wordstat_report` — расширение семантики (вложенные запросы)
- `get_search_volume_history` — история частотности по месяцам (для сезонного анализа)

### MCP Яндекс Директ (настроен в .mcp.json)
- `get_search_queries` — экспорт поисковых запросов из кампаний
- `get_keywords` — получение текущих ключевых слов кампании
- `get_campaigns` — список активных кампаний

### Python-скрипты (локальный анализ)
- `scripts/missed_demand.py` — анализ упущенного спроса из CSV
- `scripts/seasonal_forecast.py` — сезонный прогноз бюджета

Требования: Python 3.10+. Используются только стандартные библиотеки (csv, json, argparse), внешние зависимости не нужны.

### WebSearch (встроенный в Claude)
- Верификация интента через анализ поисковой выдачи

## Экспорт поисковых запросов из Яндекс Директа

### Вариант 1: Через MCP (рекомендуемый)

Вызвать MCP-инструмент `get_search_queries`:

```
get_search_queries(
  campaign_ids: [12345678],
  date_from: "2026-02-13",
  date_to: "2026-03-15",
  field_names: ["Query", "Impressions", "Clicks", "Cost", "Conversions"],
  attribution: "LYDC"
)
```

Результат сохранить как CSV для передачи в `missed_demand.py`.

### Вариант 2: Через интерфейс Яндекс Директа

1. Перейти в Яндекс Директ → Статистика → Поисковые запросы
2. Выбрать период (рекомендуется 30 дней)
3. Выбрать нужные кампании
4. Настроить столбцы: Поисковый запрос, Показы, Клики, Расход, Конверсии
5. Нажать «Экспорт» → скачать CSV/XLSX
6. Если XLSX — пересохранить как CSV (UTF-8 или Windows-1251)

### Формат CSV

Скрипт `missed_demand.py` поддерживает:
- Кодировки: UTF-8, UTF-8 BOM, Windows-1251
- Разделители: запятая, точка с запятой, табуляция
- Заголовки: русские и английские (см. маппинг в скрипте)

Пример CSV:

```csv
Поисковый запрос;Показы;Клики;Расход;Конверсии
купить кондиционер москва;1500;120;45000;8
что такое инверторный кондиционер;800;15;3200;0
кондиционер daikin цена;350;45;12000;3
```
