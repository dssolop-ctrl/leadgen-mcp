# Операторы Яндекс Wordstat

Справочник операторов для точной проверки частотности в Яндекс Вордстате и настройке ключевых слов в Яндекс Директе.

## Основные операторы

| Оператор | Пример | Что делает |
|----------|--------|-----------|
| (без оператора) | купить квартиру | Все словоформы и порядки слов. Включает запросы с дополнительными словами: «купить квартиру в москве», «квартиры купить недорого» |
| "кавычки" | "купить квартиру" | Точное количество слов (2 слова). Словоформы по-прежнему меняются: «покупка квартиры», «купил квартиру». Не включает запросы с дополнительными словами |
| !восклицательный | !купить !квартиру | Точная словоформа: только «купить» и «квартиру», не «покупка» или «квартир» |
| +плюс | купить +в москве | Принудительно учитывает предлоги и союзы (в, на, с, и, для, к), которые Wordstat обычно игнорирует |
| -минус | купить квартиру -студию | Исключает запросы, содержащие указанное слово |
| (&#x7C;) группировка | купить (квартиру&#x7C;дом) | Объединение вариантов: эквивалент двух отдельных запросов «купить квартиру» и «купить дом» |
| [...] порядок | [купить квартиру] | Фиксированный порядок слов: только «купить квартиру», не «квартиру купить» |

## Комбинирование операторов

Операторы можно и нужно комбинировать для получения точных данных.

### Примеры комбинаций

```
"!купить !квартиру"
```
Точно 2 слова + точные словоформы. Показывает только запросы из двух слов «купить квартиру» — без склонений, без дополнительных слов.

```
"купить квартиру +в москве"
```
Точно 4 слова, с обязательным предлогом «в».

```
"!купить [квартиру +в москве]"
```
Слово «купить» в точной форме, «квартиру в москве» — в фиксированном порядке.

```
купить (квартиру|дом) -студию -комнату
```
Ищет «купить квартиру» или «купить дом», исключая запросы со словами «студия» и «комната».

## Типичные ошибки

### 1. Не использовать кавычки при оценке бюджета

**Неправильно:** смотреть частотность `купить квартиру` без кавычек (покажет 500 000+, включая все вложенные запросы).

**Правильно:** проверить `"купить квартиру"` (покажет реальную частотность именно этого запроса) и `"!купить !квартиру"` (точные словоформы).

### 2. Забывать про предлоги

**Неправильно:** `"ремонт квартир москва"` — может не учесть запросы «ремонт квартир в москве».

**Правильно:** `"ремонт квартир +в москве"` — предлог «в» будет учтён.

### 3. Путать кавычки и восклицательный знак

- `"купить квартиру"` — фиксирует количество слов (2), но не словоформы
- `!купить !квартиру` — фиксирует словоформы, но не количество слов (может быть «!купить большую !квартиру»)
- `"!купить !квартиру"` — фиксирует и то, и другое

### 4. Минус-слова без проверки

Перед добавлением минус-слова проверить через Wordstat, сколько трафика оно отсечёт. Одно неверное минус-слово может убить значительную долю целевого трафика.

## Рекомендации для исследования спроса

### Этап 1: Оценка общего спроса
Используй запросы **без операторов** для понимания общего объёма рынка.

### Этап 2: Оценка реального трафика
Используй **"кавычки"** для каждой фразы, чтобы понять реальный объём (без вложенных запросов).

### Этап 3: Точная частотность
Используй **"!точные !формы"** для понимания, какие именно словоформы ищут.

### Этап 4: Группировка
Используй **(|) группировку** для быстрой проверки нескольких вариантов одной фразы.

### Этап 5: Минус-слова
Используй **-минус** для проверки, сколько трафика останется после исключения нецелевых слов.

## Применение в MCP-инструментах

При вызове `check_search_volume` через MCP передавай фразы с операторами:

```
check_search_volume(phrases: [
  "купить квартиру",        # базовая фраза в кавычках
  "!купить !квартиру",      # точные формы
  "купить квартиру",        # без кавычек (общий объём) — оператор не нужен
])
```

При вызове `get_wordstat_report` используй фразу без операторов для максимального расширения семантики.
