# Паттерны определения поискового интента

Справочник для классификации поисковых запросов по интенту. Используется при фильтрации семантики, анализе поисковых запросов и настройке минус-слов в Яндекс Директе.

---

## Коммерческий интент (COMMERCIAL)

Пользователь готов купить или выбирает конкретный товар/услугу.

**Маркерные слова:**
- купить, заказать, приобрести, оформить
- цена, стоимость, прайс, расценки, тариф
- доставка, самовывоз, курьер
- скидка, акция, распродажа, промокод
- оптом, розница, мелкий опт
- недорого, дёшево, бюджетный, экономный
- магазин, интернет-магазин, каталог
- наличие, в наличии, под заказ, срок
- гарантия, возврат, обмен

**Паттерны запросов:**
- `{товар} + купить` — купить кондиционер
- `{товар} + цена` — кондиционер цена
- `{товар} + {город}` — кондиционер москва
- `{услуга} + заказать` — заказать монтаж кондиционера
- `{товар} + недорого` — кондиционер недорого
- `{товар} + с доставкой` — кондиционер с доставкой
- `{товар} + стоимость {действия}` — стоимость установки кондиционера

**Действие в Директе:** целевые запросы, максимальный приоритет. Высокие ставки, релевантные объявления.

## Информационный интент (INFORMATIONAL)

Пользователь ищет информацию, не готов к покупке.

**Маркерные слова:**
- что такое, что это, определение, понятие
- как, каким образом, способы, методы
- зачем, почему, причины, основания
- отзывы, мнения, опыт использования
- рейтинг, топ, лучший, сравнение
- форум, обсуждение, вопрос
- своими руками, самостоятельно, diy
- бесплатно, без регистрации
- реферат, курсовая, доклад, презентация
- википедия, wiki, энциклопедия
- виды, типы, классификация, характеристики
- история, происхождение, эволюция
- схема, чертёж, план, инструкция
- плюсы и минусы, преимущества и недостатки
- принцип работы, устройство, конструкция

**Паттерны запросов:**
- `как + {действие}` — как установить кондиционер
- `что такое + {термин}` — что такое инверторный кондиционер
- `{товар} + отзывы` — кондиционер daikin отзывы
- `{товар} + своими руками` — чистка кондиционера своими руками
- `{термин} + это` — сплит-система это
- `чем отличается + {A} + от + {B}` — чем отличается кондиционер от сплит-системы
- `{товар} + рейтинг {год}` — рейтинг кондиционеров 2026

**Действие в Директе:** как правило, в минус-слова. Исключения:
- Если бизнес ведёт контент-маркетинг (блог, гайды) — можно оставить с низкими ставками
- «Отзывы» — спорный запрос; если есть страница отзывов, можно таргетировать

## Навигационный интент (NAVIGATIONAL)

Пользователь ищет конкретный сайт или бренд.

**Маркерные слова:**
- сайт, официальный сайт, офсайт
- вход, войти, авторизация, логин, пароль
- личный кабинет, аккаунт, профиль
- контакты, телефон, адрес, карта
- приложение, скачать приложение

**Паттерны запросов:**
- `{бренд} + сайт` — daikin сайт
- `{бренд} + вход` — сбербанк онлайн вход
- `{бренд} + личный кабинет` — мегафон личный кабинет
- `{бренд} + контакты` — леруа мерлен контакты
- `{бренд} + {город}` — может быть и коммерческим

**Действие в Директе:**
- Чужие бренды → в минус-слова (кроме случаев осознанного конкурентного таргетинга)
- Свой бренд → брендовая кампания с низкими ставками (защита от конкурентов)

---

## Транзакционный интент (TRANSACTIONAL)

Пользователь хочет выполнить конкретное действие онлайн.

**Маркерные слова:**
- скачать, download, загрузить
- зарегистрироваться, регистрация, создать аккаунт
- подписаться, оформить подписку
- рассчитать, калькулятор, онлайн-расчёт
- записаться, забронировать, оформить
- оставить заявку, получить консультацию
- онлайн, online

**Паттерны запросов:**
- `рассчитать + {услугу}` — рассчитать стоимость ремонта
- `{действие} + онлайн` — оформить страховку онлайн
- `калькулятор + {продукт}` — калькулятор ипотеки
- `записаться + {куда}` — записаться к стоматологу
- `заявка + на + {услугу}` — заявка на кредит

**Действие в Директе:** высокий приоритет, если сайт позволяет выполнить действие. Посадочная страница должна содержать форму/калькулятор/сервис.

## Матрица решений

| Интент | Приоритет | Действие в Директе | Ставка | Посадочная |
|--------|-----------|-------------------|--------|-----------|
| COMMERCIAL | Высокий | Таргетировать | Высокая | Товар/услуга с CTA |
| TRANSACTIONAL | Высокий | Таргетировать | Высокая | Форма/калькулятор/сервис |
| INFORMATIONAL | Низкий | Обычно в минус | - | Статья (если контент-маркетинг) |
| NAVIGATIONAL (свой бренд) | Средний | Брендовая кампания | Низкая | Главная/каталог |
| NAVIGATIONAL (чужой бренд) | Нет | В минус-слова | - | - |
| MIXED | Средний | Тест с мониторингом | Средняя | Зависит от контента |

## Как определить интент неоднозначного запроса

Если запрос не попадает ни в одну категорию однозначно:

1. **Проверить через Wordstat** — посмотреть вложенные запросы. Если большинство вложенных коммерческие — запрос коммерческий.

2. **Проверить выдачу** — использовать workflow «Верификация интента» из skill.md. Если ТОП-10 — магазины и лендинги, интент коммерческий.

3. **Проверить CTR и конверсии** — если запрос уже есть в кампании, посмотреть метрики. Высокий CTR + конверсии = коммерческий.

4. **Правило по умолчанию:** если сомневаешься — добавить с мониторингом. Через 2 недели проверить: есть конверсии → оставить, нет → в минус.

## Регулярные выражения для автоматической классификации

Эти паттерны используются в `scripts/missed_demand.py`:

**Информационный (в минус-слова):**
```
что\s+такое, как\s+, зачем, почему, отзыв[ыа]?, рейтинг,
сравнени[ея], лучши[йех], форум, своими\s+руками, бесплатно,
скачать, реферат, курсовая, википедия, wiki, чем\s+отличается,
виды, типы, классификаци[яю], истори[яю], определение,
примеры?, инструкци[яю], схема, чертеж, самостоятельно, diy
```

**Коммерческий (оставить):**
```
купить, заказать, цена, стоимость, прайс, доставк[аи],
скидк[аи], акци[яи], оптом, недорого, магазин, каталог,
в\s+наличии, гарантия
```
