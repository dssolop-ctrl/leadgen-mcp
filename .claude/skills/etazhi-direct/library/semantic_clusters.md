# Семантические кластеры — Этажи

Базовые ключевые слова по тематикам из рабочих таблиц. Используй как стартовую точку при создании кампании.

> **Обязательно:** расширяй через `wordstat_top_requests` для конкретного города.
> **Важно:** в ключи вставляй название города (например, «купить квартиру Тюмень»).
> Минус-слова для каждого кластера — в `library/negative_keywords.md`.

## Правила работы с кластерами

1. **{город} / ГОРОД** — заменяй на конкретный город и его варианты (Тюмень / в Тюмени / тюменский)
2. **Расширяй** через `wordstat_top_requests(keywords="<базовый запрос>")` — реальные хвосты
3. **Частотность** — проверяй `check_search_volume(keywords=[...])` перед добавлением
4. **Минус-слова** — обязательно из секции минусов + из `library/negative_keywords.md`
5. **Не дублируй** — используй `deduplicate_keywords` при добавлении в разные группы
6. **Интент** — «купить», «цена» = транзакционный; «что такое», «отзывы» = информационный → в минус
7. **Покупатель vs Продавец** — 95% кампаний на покупателя. Продавец — отдельная кампания
8. **Конкуренты в ключах** — авито, циан, домклик — конкурентные запросы, высокий интент покупателя
