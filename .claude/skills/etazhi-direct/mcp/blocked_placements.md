## Как обновлять

1. Получить отчёт по площадкам из интерфейса Директа (или через будущий MCP-инструмент)
2. Площадки с расходом > 500₽ и 0 конверсий за 30 дней → в чёрный список

4. Обновить этот файл
5. Применить ко всем РСЯ-кампаниям через интерфейс
