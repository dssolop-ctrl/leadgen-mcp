# Минус-слова — Этажи

Типовые минус-слова для кампаний недвижимости.
Применяются на уровне кампании при создании.

> При создании кампании — загрузи соответствующий блок.
> При оптимизации — проверь поисковые запросы и дополни.

## Общие (для ВСЕХ кампаний недвижимости)

### Информационные
```
что такое, как выбрать, советы, рейтинг, обзор, сравнение, отзывы, форум,
своими руками, самостоятельно, бесплатно, скачать, реферат, курсовая,
википедия, определение, виды, типы, классификация, история, схема,
чертёж, план, инструкция, принцип работы
```

### Нерелевантные услуги
```
ремонт, дизайн интерьера, уборка, клининг, переезд, грузчики,
мебель, техника, обои, плитка, ламинат, сантехника
```

### Работа / Вакансии (если не HR-кампания)
```
вакансии, работа, зарплата, оклад, резюме, трудоустройство, риэлтор вакансия
```

### Юридические / Информационные
```
закон, судебная практика, наследство, приватизация, прописка,
регистрация, выписка, документы для, оформление прав
```
