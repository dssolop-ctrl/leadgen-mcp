# Настройка Yandex XML API

## Вариант 1: Бесплатный Yandex XML (рекомендуется для начала)

Бесплатный доступ к поисковой выдаче Яндекса. Квота: 10 запросов/день, до 1000/день после верификации.

### Шаги:

1. Перейдите на **https://xml.yandex.ru/**
2. Войдите с аккаунтом Яндекса
3. Нажмите «Зарегистрироваться» для получения доступа к XML Search
4. После регистрации на странице профиля найдите:
   - **Логин** (ваш Яндекс-логин)
   - **API-ключ** (длинная строка)
5. Добавьте в `tokens.env` в корне проекта:

```env
YANDEX_XML_USER=ваш_логин
YANDEX_XML_KEY=ваш_api_ключ
SITE_DOMAIN=ваш-сайт.ru
```

### Квота:

| Уровень | Запросов/день |
|---------|---------------|
| Базовый (после регистрации) | 10 |
| После подтверждения сайта | 100 |
| После верификации | до 1000 |

**Совет:** 10 запросов/день достаточно для еженедельной проверки 5-10 ключевых запросов. Не расходуйте на тесты.

## Вариант 2: Yandex Cloud Search API (платный, больше квоты)

Платный API через Yandex Cloud. Подходит для ежедневного мониторинга большого числа запросов.

### Шаги:

1. Зарегистрируйтесь в **Yandex Cloud**: https://cloud.yandex.ru/
2. Создайте каталог (folder) в консоли
3. Получите OAuth-токен: https://oauth.yandex.ru/authorize?response_type=token&client_id=1a6990aa636648e9b2ef855fa7bec2fb
4. Скопируйте ID каталога из консоли Yandex Cloud
5. Добавьте в `tokens.env`:

```env
YANDEX_CLOUD_OAUTH_TOKEN=ваш_oauth_токен
YANDEX_CLOUD_FOLDER_ID=ваш_folder_id
```

6. Для получения IAM-токена (автоматически кешируется на 11 часов):

```bash
scripts/iam_token.sh
```

### Стоимость:

Актуальные цены: https://cloud.yandex.ru/docs/search-api/pricing

## Проверка настройки

После заполнения `tokens.env` проверьте работу:

```bash
# Тестовый запрос
scripts/search.sh --query "тест" --region 213
```

Если видите таблицу с результатами — всё настроено правильно.

Частые ошибки:
- **Ошибка 18** — неверный логин или ключ, проверьте `YANDEX_XML_USER` и `YANDEX_XML_KEY`
- **Ошибка 15** — квота исчерпана, попробуйте завтра
- **Ошибка 401/403** — проблема с авторизацией, пересоздайте ключ на xml.yandex.ru
