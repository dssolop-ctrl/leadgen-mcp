# Уточнения (Callouts)

Библиотека уточнений для объявлений Этажи. Макс. 25 символов на уточнение, до 4 на объявление.
Данные собраны из 4 городов: Тюмень, Краснодар, Омск, Сургут (~80 уникальных текстов).

> **При создании объявления** (Шаг C6):
> 1. Выбери 4 релевантных уточнения из секции «✅ Разрешены» твоей тематики.
> 2. ОБЯЗАТЕЛЬНО сверься с секцией «❌ ЗАПРЕЩЕНЫ» — не используй эти тексты.
> 3. Сверься с разделом «Универсальные запреты (везде)» — эти фразы запрещены всегда.
> 4. Если уточнение не в текущей теме, но по смыслу подходит — проверь, нет ли его
>    в deny-list другой темы (это индикатор, что оно рисковое).

---

## 🚫 Универсальные запреты (ВЕЗДЕ, любая тематика)

Эти фразы НИКОГДА не ставим в уточнения, независимо от тематики:

| ❌ Текст | Почему запрещено |
|---|---|
| Рассрочка 0% | Сложно проверить применимость к конкретному объекту — риск рекламации |
| Рассрочка 0% до 3 лет | То же |
| Есть рассрочка | Неопределённо, требует подтверждения по каждому объекту |
| Гарантия низких ставок | Обещание, которое сложно подтвердить — риск ФАС |
| Лучшие цены | Сравнительные превосходные формы — риск ФАС |
| Самые низкие | То же |
| №1 на рынке | Требует документального подтверждения |
| Выгоднее всех | Сравнительная форма без базы сравнения |

> См. также общие запреты в `LEGAL.md` (корень проекта).

## Рекомендации по выбору (после allow/deny проверки)

1. **Поиск** — акцент на безопасность + УТП, разрешённые на теме (без комиссии, рассрочка, просмотр — только где разрешены)
2. **РСЯ** — акцент на выгоду + доверие (скидки, №1 в России, большая база)
3. **Не дублировать** текст объявления — уточнения дополняют, не повторяют
4. **Сезонные** — «Семейная ипотека», «Маткапитал» актуальны при действующих программах, уточнять в ипотечных кампаниях

## Процедура проверки уточнений в Шаге C6 (обязательная)

```
1. Выбрал 4 уточнения из секции «✅ Разрешены» текущей темы.
2. Для КАЖДОГО уточнения проверь:
   a) Не в разделе «🚫 Универсальные запреты (ВЕЗДЕ)»?
   b) Не в «❌ ЗАПРЕЩЕНЫ» текущей темы?
   c) Не в «❌ ЗАПРЕЩЕНЫ» другой темы (если да — повышенный риск,
      уточни у пользователя перед использованием).
3. Если хоть одно не прошло — ЗАМЕНИ на альтернативу из «✅ Разрешены».
4. Перед add_ad_extension — финальная юрпроверка по LEGAL.md.
```
