# Copy Blacklist — запрещённые слова в текстах Директа

Слова и фразы, которые нельзя использовать в любых текстовых полях объявлений и общих ассетах (быстрые ссылки, уточнения, callouts).

## Проверяемые поля

- `Title` (заголовок 1)
- `Title2` (заголовок 2)
- `Text` (текст объявления)
- `SitelinkSet.Title` + `SitelinkSet.Description` (быстрые ссылки)
- `Callout.Value` (уточнения)
- `DisplayUrlPath` (отображаемый URL)

## Жёсткий blacklist

| Слово / фраза | Почему запрещено | Допустимая замена |
|---|---|---|
| `WhatsApp`, `Whatsapp`, `ватсап` | Запрещено в Директе (модерация) | «быстрая связь», «связь с менеджером», «онлайн-консультация» |
| `VPN`, `ВПН`, `впн` | Запрещено | — (полный запрет, не нужно в недвижке) |
| `официальный сайт` | Может ввести в заблуждение (претензии от правообладателей) | «официальные предложения», «прямые продажи», конкретный бренд |
| `без СМС`, `без смс`, `без смс и регистрации` | Наследие пиратского контента, модерация банит | «быстрая обработка заявки», «моментальный отклик» |
| `бесплатная консультация` без указания условий | ФАС может счесть за скрытую плату | «консультация по объектам» + честная цена услуги |

## Мягкий blacklist (требует контекста)

| Слово / фраза | Когда запрещено | Альтернатива |
|---|---|---|
| `гарантия` | В контексте обещания цены/сроков | «фиксированные условия», «прописано в договоре» |
| `100%`, `гарантированно` | Про конверсию / продажу | «высокий шанс», «практика показывает» |
| `лучший`, `№1`, `топ-1` | Без подтверждения (рейтинг / награда) | «входим в топ», «один из лидеров» с указанием источника |
| `дешевле всех`, `минимальная цена` | Без проверки | «выгодные предложения», «цены от X ₽» |
| `абсолютно`, `максимально` | Часто триггерит модерацию | убрать или заменить на конкретику |

## Проверка shared assets

**Важно:** после обновления одного объявления — проверить, не содержит ли запрещённое слово shared `SitelinkSet` или `CalloutSet`, привязанный к десяткам других объявлений.

**Паттерн исправления shared набора:**
1. Создать новый `SitelinkSet` / `CalloutSet` без запрещённых слов
2. Bulk-update всех `ad_ids`, ссылающихся на старый set — `update_ad(sitelink_set_id=<new>)`
3. Read-back: `get_ads` с фильтром по старому set → `count == 0`
4. Старый set оставить, если на него ссылается только архивный хвост

## Автопроверка

Запускать перед `moderate_ads`:

```bash
python3 .claude/skills/leadgen/scripts/check_copy.py --campaign-id <id>
```

Или перед созданием — прогнать черновики через скрипт со списком из TSV.

## Как пополнять этот файл

- После любого отказа модерации с формулировкой «запрещённая формулировка» — добавить строку в жёсткий blacklist
- После жалобы ФАС / бренда — добавить в мягкий blacklist
- После обновления LEGAL.md — синхронизировать
