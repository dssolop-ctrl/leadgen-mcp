# Структура сайта etagi.com — посадочные страницы

URL-дерево для выбора посадочной страницы при создании кампании.
Домен города: `{city}.etagi.com` (Тюмень = `www.etagi.com`).

> При создании кампании: выбери посадочную из таблицы ниже по тематике.
> Фильтры через параметры URL добавляются по необходимости.

---

---

## Правила выбора посадочной

1. **[site]** — основной сайт `{city}.etagi.com` + URL из таблицы выше
2. **[LP]** — лендинг `etagi.net` — только для промо-акций или если указан специалистом
3. Для города с поддоменом: `krasnodar.etagi.com/realty/`, `spb.etagi.com/zastr/` и т.д.
4. Тюмень = `www.etagi.com` (без поддомена)
5. При выборе — максимально специфичная страница (дома → `/realty_out/doma/`, не `/realty_out/`)
