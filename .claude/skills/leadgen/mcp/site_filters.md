# Фильтры сайта etagi.com — параметры URL для посадочных

> **Статус:** ЗАПОЛНЯЕТСЯ. Этот файл описывает, какие фильтры поддерживает сайт через URL-параметры.
> Используется при создании посадочных URL для гео-кластеров и других уточнений.

---

## Известные параметры

| Параметр | Значения | Пример URL | Описание |
|----------|----------|------------|----------|
| `rooms` | 1, 2, 3 | `/realty/?rooms=1` | Фильтр по количеству комнат |
| `installment` | `!` | `/zastr/?installment=!` | Новостройки с рассрочкой |

---

## Фильтры по районам/микрорайонам — ТРЕБУЕТ ИССЛЕДОВАНИЯ

**Вопросы для исследования:**
1. Поддерживает ли сайт фильтр по району через URL-параметр? (напр. `?district=...`, `?area=...`)
2. Какие ID/slug используются для районов в каждом городе?
3. Есть ли карточные/табулярные страницы районов (напр. `/realty/rayon-neftyaniki/`)?
4. Поддерживается ли комбинация фильтров (`?rooms=1&district=neftyaniki`)?

**Как исследовать:**
- Проверить URL при выборе района в фильтре на сайте
- Проверить наличие ЧПУ-страниц по районам
- Проверить API каталога (если доступен)

**До получения данных:** использовать базовый URL без гео-фильтра (напр. `/realty/?rooms=1`).
Гео-уточнение идёт только через заголовок и ключевые слова объявления.

---

## Другие потенциальные фильтры

| Параметр | Значения | Статус |
|----------|----------|--------|
| Цена от/до | `?price_from=...&price_to=...` | Не проверено |
| Площадь | `?area_from=...&area_to=...` | Не проверено |
| Этаж | `?floor_from=...` | Не проверено |
| Тип сделки | `?deal_type=...` | Не проверено |

---

> **Правило:** если фильтр не подтверждён — НЕ использовать в посадочных URL.
> Битый фильтр в URL может привести к 404 или пустой выдаче, что убьёт конверсию.
