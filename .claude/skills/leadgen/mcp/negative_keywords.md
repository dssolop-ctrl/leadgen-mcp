# Минус-слова — Этажи

Типовые минус-слова для кампаний недвижимости.
Применяются на уровне кампании при создании.

> **Для рантайма используй MCP-инструмент** `get_negative_keyword_guidance(theme, city?, placement?, include_competitors?)`. Он возвращает все блоки ниже + готовую строку через запятую для `negative_keywords` и хард-гейт ≥150 слов. **Этот файл — только для ручного просмотра** (если нужно глазом сверить полный список).
>
> При создании кампании в скилле — НЕ читай файл целиком, зови MCP. При оптимизации — добери блоки через тот же MCP-вызов.

## Чужие города РФ (ОБЯЗАТЕЛЬНО для каждой региональной кампании)

> Добавляй в `negative_keywords` кампании ВСЕ крупные города РФ и Казахстана,
> кроме собственного, а также очевидных дублей (омск ≠ омский → минусуем только лишние формы).
>
> Исключения:
> - Твой город + его варианты («омский», «в омске») — оставляем!
> - Города, где кампания тоже крутится (межрегиональные) — исключить из списка.
>
> Зачем: без этого блока запросы вида «купить квартиру москва» или «квартира спб»
> могут сработать в кампании на Омск, потому что геотаргетинг Директа работает
> по IP/локации пользователя, а не по тексту запроса. Приезжий из Москвы,
> ищущий московскую квартиру с Омска, — прямая утечка бюджета.

## Шаблон сборки минус-слов для кампании (ОБЯЗАТЕЛЬНО)

Для ЛЮБОЙ региональной кампании собирай минус-слова по следующему шаблону:

| # | Блок | Примерно слов |
|---|---|---|
| 1 | Чужие города РФ (все крупные кроме собственного + его словоформ) | ~80 |
| 2 | Международные и зарубежная недвижимость | ~25 |
| 3 | Информационные — базовые | ~40 |
| 4 | Бесплатно / скачать / обучение | ~25 |
| 5 | Нерелевантные услуги (ремонт, мебель, стройка) | ~25 |
| 6 | Работа / Вакансии (если не HR-кампания) | ~12 |
| 7 | Юридические / бюрократия (если не юркампания) | ~20 |
| 8 | Медицина / семья / пересечения | ~20 |
| 9 | Негатив / жалобы | ~10 |
| 10 | Тематические (из блока конкретной темы) | ~20-30 |
| 11 | Конкуренты-агрегаторы (если не в ключевых) | ~20 |
| 12 | Конкуренты-агентства (если не брендовая) | ~15 |
| — | **ИТОГО базовый чек** | **≥ 250** |

### Хард-гейт перед `add_campaign`

Итоговое число минус-слов кампании **должно быть ≥ 150**. Если меньше:

1. Проверь, не пропущен ли блок «Чужие города РФ»
2. Проверь, не пропущен ли блок «Информационные»
3. Вернись в `mcp/negative_keywords.md` и добери недостающие блоки

### Техническое

- Яндекс Директ API v5 лимит: **4096 минус-слов** на кампанию, **суммарно до 20000 символов**.
  При 250 словах среднего размера (~15 символов) итоговая строка ~4000 символов — хватает с запасом.
- Минус-слова добавляются на уровень КАМПАНИИ при `add_campaign` через `negative_keywords="..."`.
  Минусы на уровне групп — только для кросс-минусации (см. Шаг C8).
- В `negative_keywords` передавай строку через запятую: `"москва, спб, циан, ..."`.
