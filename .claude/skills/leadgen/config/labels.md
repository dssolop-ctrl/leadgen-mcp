# Метки кампаний (Labels) — Этажи

Метки (теги) присваиваются каждой кампании для агрегации статистики в системе аналитики.

> **ОБЯЗАТЕЛЬНО:** присвоение меток — обязательный шаг при создании кампании, наравне с подключением счётчика и указанием целей. Без меток статистика по кампании не попадёт в отчёты.

## MCP-инструменты для меток

Метки управляются через **API v4 Live** (в API v5 нет сервиса Labels). Доступные инструменты:

| Инструмент | Описание |
|---|---|
| `get_labels` | Получить текущие метки кампаний (GetCampaignsTags) |
| `set_campaign_labels` | Установить метки на кампании (**заменяет** все текущие) |
| `add_labels` | Добавить метки к существующим (get → merge → update) |

**После создания кампании** — вызови `add_labels` с метками из таблицы ниже.

## Справочник меток

## Как определить метки при создании кампании

1. Определи **тематику** из названия кампании (Вторичка, Загородка, Новостройки и т.д.)
2. Определи **направление** — Покупатель или Продавец (по умолчанию Покупатель)
3. Найди строку в таблице выше
4. Присвой ВСЕ метки из столбца tags
5. Для имиджевых/медийных кампаний — метка `Имидж`
6. Для HR — метка `HR`
7. Для тестовых — добавь метку `Тест`

### Примеры

| Название кампании | Метки |
|---|---|
| Тюмень \| Поиск \| Вторичка \| Однокомнатные \| [site] | `Лидген, Вторичка, Покупатель` |
| Тюмень \| РСЯ \| Загородка (покупатель) \| Дома [site] | `Лидген, Загородка, Покупатель` |
| Тюмень \| Мастер кампаний \| Новостройки (общие) \| Рассрочка \| [site] | `Лидген, Новостройки, Общие` |
| Тюмень \| Медийная \| Имидж \| Ипотека чаще одобряется \| [site] | `Имидж` |
| Краснодар \| РСЯ \| HR \| [site] | `HR` |
| Тюмень \| Мастер кампаний \| Межрег \| ЗАГОРОДКА \| [site] | `Лидген, Межрег` |
