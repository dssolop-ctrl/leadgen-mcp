# leadgen-mcp

Self-hosted MCP-сервер для управления рекламой в Яндекс Директе и VK Ads через AI-агентов. Включает специализированные скиллы для Claude Code: создание кампаний, сбор семантики, аналитика и оптимизация.

## Платформы

| Платформа | Инструментов | Возможности |
|-----------|-------------|-------------|
| Яндекс Директ | ~50 | Кампании, группы, объявления, ключевые, метки (API v4+v5) |
| Яндекс Метрика | ~11 | Отчёты, цели, конверсии, UTM-аналитика |
| Wordstat | 5 | Частотность, SearchedWith-расширение, динамика, регионы |
| VK Ads | ~30 | Кампании, таргетинг, ретаргетинг, контекстные фразы |
| Reference tools | 6 | Конфиги городов, UTM, семантические кластеры, площадки |

## Скиллы для Claude Code

Скиллы в `.claude/skills/` — workflow-гайды, которые оркестрируют MCP-инструменты:

| Скилл | Описание |
|-------|----------|
| `etazhi-direct` | Создание, анализ и оптимизация кампаний недвижимости в Яндекс Директе |
| `vk-ads` | Управление рекламой VK Ads: кампании, таргетинг, ретаргетинг |
| `yandex-metrika` | Аналитика: отчёты по UTM, CPA, конверсии, сравнение периодов |
| `demand-research` | Исследование спроса: упущенный спрос, каннибализация, сезонность |
| `serp-monitor` | Мониторинг позиций в выдаче Яндекса, анализ конкурентов |

### Ключевые возможности workflow `etazhi-direct`

- **Многораундовый сбор семантики** — seed-матрица из кластеров → расширение через Wordstat SearchedWith → 30–80 фраз вместо 10–15
- **Кластеризация → мульти-группы** — семантика автоматически разбивается на кластеры (гео-районы, интент, условия), каждый кластер → отдельная группа объявлений с индивидуальными заголовками
- **Гео-кластеры в объявлениях** — название района в заголовке + фильтр в посадочной URL
- **Reference tools** — конфиги городов, UTM, площадки, сайтлинки, семантические кластеры доступны через MCP без загрузки файлов в контекст
- **Метки через API v4 Live** — назначение на объявления (отображаются на уровне групп)

## Быстрый старт

```bash
cd server
cp config.yaml.example config.yaml
# Заполните токены в config.yaml
docker build -t leadgen-mcp .
docker run -p 8080:8080 -v ./config.yaml:/app/config.yaml leadgen-mcp
```

## Структура

```
server/                  # Go MCP-сервер
  platform/
    direct/              # Яндекс Директ API v5 + v4 (метки)
      references.go      # Reference tools (города, UTM, семантика)
      references_data.go # Embedded data (34 города, 9 кластеров, 502 площадки)
      labels.go          # Метки — API v4 Live (banner-level)
    metrika/             # Яндекс Метрика API
    wordstat/            # Яндекс Wordstat API v4
    vk/                  # VK Ads API v2
.claude/skills/          # Скиллы для Claude Code
  etazhi-direct/         # Основной скилл: создание и оптимизация кампаний
    skill.md             # Workflow: 9 шагов создания кампании
    config/              # Именование, метки, UTM
    library/             # Заголовки, тексты, сайтлинки, уточнения
    references/          # Структура сайта, фильтры, бенчмарки
campaigns/               # Файлы кампаний (история, статус, гипотезы)
```
